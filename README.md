# Concept

A Space Invaders-like pygame project to train my abilities with OOP, done
following the structure shown in  "Python Crash Course" book from Eric Matthes. 

# Elements

The ship that the player moves, the aliens (grouped in a fleet) the player
has to shoot down, the bullets for killing the aliens, and a powerup that
spawns recurringly that increases the quantity of the bullets shot.

# Commands

Q - Quits the game

Left/Right Arrows - Movement

Spacebar - Shoot bullet
